## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

# Instalação e execução modo DEV

## API e Micro serviços

```bash
$ npm install
$ npm run start:dev
```

## Frontend

### em modo dev ja esta configurado o proxy para rodar a api na porta 3000

```bash
$ yarn
$ yarn run dev
```
